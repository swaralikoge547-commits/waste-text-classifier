# Waste text dataset schema

item_name: short item title (string)  
description: natural language description of the waste item (string)  
category: one of {wet, dry, recyclable, hazardous, e-waste} (string, label)  
disposal_guideline: 1–2 line instruction for user (string)  
source: origin of example (synthetic / real_user / faq) (string, optional)  
notes: internal notes on edge cases (string, optional)
